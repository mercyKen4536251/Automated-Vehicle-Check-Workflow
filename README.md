# 自动化车辆审核工作流

基于VLM模型的自动化车辆审核系统，通过5个智能体节点进行多维度图像审核。

## 快速开始
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 核心功能

### 🧩 提示词管理
管理5个节点对应的审核提示词，支持版本管理（创建、更新、激活）。

### 🖼️ 参考图库管理
为每个车系添加标准参考图（最多5张），用于视角和细节一致性比对。

### 📋 测试用例管理
管理goodcase和badcase，支持问题标签标注和多维度筛选。

### 🚀 运行中心
批量执行测试用例，支持多线程并发和实时进度显示。

### 📊 结果面板
查看测试准确率、详细结果和历史记录。

## 使用流程

1. **配置模型** - 在"配置管理"中添加火山引擎API Key
2. **设置提示词** - 为5个节点配置并激活提示词版本
3. **添加参考图** - 为各车系上传标准参考图
4. **创建用例** - 添加测试用例并标注问题标签
5. **执行测试** - 在运行中心筛选并执行测试
6. **分析结果** - 在结果面板查看通过率和失败原因

## 审图流程
1. Node1: 汽车检测 (car="yes")
2. Node2: 裁切检测 (cropping="no")
3. Node3: 运动/无人驾驶检测 (match="yes")
4. Node4: 视角一致性 (match="yes")
5. Node5: 细节一致性 (match="yes")
